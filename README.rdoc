== Body Beast Tracker
{<img src="https://travis-ci.org/DocOnDev/beast.svg?branch=master" alt="Build Status" />}[https://travis-ci.org/DocOnDev/beast] {<img src="https://codeclimate.com/github/DocOnDev/beast/badges/gpa.svg" />}[https://codeclimate.com/github/DocOnDev/beast]


Meal tracking for Body Beast program through Beach Body.

Uses Rails 4.2.x

===Setup:
* Clone repository

* bundle install

* rails server

===Tests:
* bundle exec rake test

* bundle exec rake cucumber
